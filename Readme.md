# Task-4

## Project Overview

This project is a simple webpage created using HTML and CSS. It displays an image inside a container along with a clickable button placed below the image. The clickable button is customised with the help of Cascadig style sheet.

## Features

* Structured HTML document with external CSS.
* Image displayed inside a container.()
* Gradient-styled button.
* Equal spacing of 25% from the left and right sides of the screen.
* Equal spacing of 10% from the top and bottom of the screen.
* No Flexbox used.
* No `position: relative` or `position: absolute`.
* No `overflow: hidden`.

## Technologies Used

* HTML5
* CSS3

## Deployed link 

The webpage displays:

* A centered image card.
* A gradient-colored button below the image.
* Proper spacing around the content according to the specified requirements.
